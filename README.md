# DBDMobile_API_Tracker
<img src="https://i.imgur.com/zVt0gAc.png" align="right" alt="Wwise Logo" width="256" height="256">

This repository is used to track changes made in Dead By Daylight Mobile API traffic.<br>
I try to keep this repository updated at least once a day, each update is pushed manually.

**List of tracked builds:**
* Live -> Live Build
* DEV -> Developer Build
* QA -> Quality Assurance Build

# What Should You Look For?
There's a lot information you can obtain. Developers are unfortunately aware of this, so they add bare minimum, most of the times 30 minutes before release. Let me give you just some examples:
- New Characters
- New Cosmetics
- Cosmetics Release Dates
- Cosmetics Price
- Bloodhunts
- UI Assets

and way more..
